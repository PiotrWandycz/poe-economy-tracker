# Prototype: economy overview UI variants

**Question**: What layout works best for showing multiple Sections of economy data on one page?

**Winner: Variant B — Card grid**

Each Section as a card in a responsive masonry grid, colour-coded by Group (General = blue, Atlas = purple). Item name + Value per row inside each card.

**Discard**: Variant A (dense tables) and Variant C (accordion).
